# Saúde Open Source Brasil (Edição 2026)

Uma lista curada de projetos, plataformas e comunidades de código aberto na área da saúde em desenvolvimento no Brasil. 

O objetivo deste repositório é centralizar e divulgar iniciativas que utilizam a tecnologia de forma aberta para resolver problemas da saúde no país, facilitando a colaboração entre desenvolvedores, pesquisadores e profissionais da área.

---

## Análise de Dados e Epidemiologia

Projetos focados no processamento e análise de dados epidemiológicos, demográficos e de saúde pública, alinhados ao **Plano de Dados Abertos do Ministério da Saúde (2024-2026)**.

* **[PCDaS - Plataforma de Ciência de Dados aplicada à Saúde (Fiocruz)](https://github.com/pcdas)**: Uma das infraestruturas mais robustas do país para análise de grandes volumes de dados do SUS, oferecendo ferramentas e datasets integrados.
* **[Instituto Todos pela Saúde (ITpS)](https://github.com/InstitutoTodosPelaSaude)**: Referência em vigilância genômica e epidemiológica, com foco em arboviroses e monitoramento de variantes virais.
* **[Observatório de Saúde na Infância (Observa Infância)](https://github.com/fiocruz/observa-infancia)**: Monitoramento de indicadores de saúde infantil baseado em dados abertos.
* **[covid19br / vac](https://github.com/wcota/covid19br)**: Projeto histórico de consolidação de dados de COVID-19 que evoluiu para modelos de análise de cobertura vacinal sistemática.
* **[Base dos Dados (Saúde)](https://basedosdados.org/)**: Repositório que facilita o acesso a tabelas do DATASUS (SIM, SINASC, SIH) tratadas e prontas para análise em SQL/Python/R.

## Inteligência Artificial e Modelos de Linguagem (LLMs)

Novas iniciativas focadas na aplicação de IA Generativa e modelos preditivos no contexto clínico brasileiro.

* **[Maritaca AI - Sabiá](https://github.com/Maritaca-AI)**: Embora seja um modelo de linguagem genérico, o Sabiá é amplamente utilizado pela comunidade brasileira para *fine-tuning* em contextos médicos em português.
* **[Clinical-BERT-PT](https://github.com/pucrs-nlp)**: Modelos de processamento de linguagem natural (NLP) treinados especificamente em prontuários e textos médicos em português do Brasil.
* **[IA para Gestão de Leitos](https://github.com/ImpulsoGov)**: Projetos que utilizam algoritmos preditivos para otimizar filas e disponibilidade de leitos no SUS.

## Bioinformática e Pesquisa Genômica

Projetos e ferramentas para análise de dados biológicos, essenciais para a medicina de precisão brasileira.

* **[Bioinformatics-INCA](https://github.com/bioinformatics-inca)**: Scripts e pipelines para genômica do câncer.
* **[Bioinfo Brasil](https://github.com/lubianat/bioinfo_brasil)**: Mapeamento da rede de pesquisadores e ferramentas de bioinformática no ecossistema nacional.
* **[Grupo de Estudo de Bioinformática - PyLadies Brasil](https://github.com/pyladies-brazil/grupo-estudo-bioinformatica)**: Comunidade ativa na produção de material didático e tutoriais open source.

## Interoperabilidade e Padrões (RNDS/FHIR)

Projetos focados na troca de informações em saúde, seguindo os padrões exigidos pela **Rede Nacional de Dados em Saúde**.

* **[FHIR Brasil](https://github.com/FHIR-Brasil)**: Implementações de referência e guias de IG (Implementation Guides) para o perfil brasileiro de interoperabilidade.
* **[openEHR Brasil](https://openehr.org.br/)**: Focado na semântica de dados e modelagem de arquétipos para registros eletrônicos de saúde.
* **[ConnectSUS / RNDS Utils](https://github.com/ministeriodasaude)**: Repositórios (governamentais e de comunidade) que auxiliam desenvolvedores na integração com as APIs da RNDS.

## Prontuários e Gestão (APS)

Soluções voltadas para a Atenção Primária à Saúde e gestão hospitalar.

* **[Impulso Previne](https://github.com/impulso-gov/impulso-previne)**: Ferramenta open source que auxilia municípios brasileiros no monitoramento dos indicadores do Previne Brasil.
* **[GNU Health Brasil](https://github.com/gnuhealth-br)**: Adaptação brasileira do sistema internacional de gestão hospitalar livre.
* **[PEC-PostgreSQL Interface](https://github.com/portalsds/sigtap-postgresql)**: Ferramentas para extração e integração de dados do Prontuário Eletrônico do Cidadão (e-SUS APS).

## Como Contribuir

A comunidade open source em saúde no Brasil cresce rápido! Se você conhece um novo projeto em 2026:

1.  **Fork este repositório.**
2.  **Crie uma branch:** (`git checkout -b minha-contribuicao`).
3.  **Adicione o projeto:** Edite o `README.md`.
4.  **Faça o commit:** (`git commit -m 'Adiciona novo projeto de IA na saúde'`).
5.  **Push e Pull Request.**
